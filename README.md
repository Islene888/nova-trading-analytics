## Project Overview

This repository documents the quantitative analytics, operations research models, and data engineering work performed by the E-Commerce Operations Research Analyst at Nova Trading Inc.

Nova Trading Inc. operates multi-channel e-commerce retail across:
- **Shopify** — independent direct-to-consumer website
- **Amazon Marketplace** — third-party marketplace sales
- **TikTok Shop** — social commerce platform

The analytical work in this repository supports data-driven decision-making across all channels, including demand forecasting, pricing optimization, customer behavior modeling, A/B testing, and cross-channel attribution analysis.
